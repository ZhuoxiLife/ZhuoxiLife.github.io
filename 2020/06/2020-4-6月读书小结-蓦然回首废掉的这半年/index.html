2020 4-6月读书小结 - 蓦然回首，废掉的这半年

2020-06-22 发布在 读书, 读书小结

三月份Covid19 登陆美国，大概有百例，国内的疫情开始好转了，那时渐渐的从担心家人到担心自己，有一种一面放心又一面焦躁的感觉。 今天是7月4日， 目前美国病例数是2,860,000。现在已经能渐渐适应new normal。在家工作的四个月，可谓是极其没有效率，无论是工作，学习，还是生活。不是没有目标， 不是没有环境，就是莫名其妙的没有动力。现在想想，当时的情绪已经没了，忽然有点不可思议，怎么这大好时光就这么浪费掉了？

四月读了1本书，五月读了2本书，六月中旬读了几本书，下了一个APP，稍微有点气色了，希望能延续到下半年。

情绪管理

不知不觉已经是用weobot第28天了 Woebot，是斯坦福大学推出的一个心理治愈聊天机器人。多亏了它， 我每天上午10点左右睁开眼睛，百无聊赖躺在床上，跟它聊一分钟，就能爬起来了。它使用的治疗方法叫“认知行为治疗”。 这种治疗的根据是想法， 情绪，行为之间会相互影响。关于此类治疗方法的书很多很多， 这个app的精妙之处在于他把拆散成一个个的点， 然后以每天聊天的形式讲出来，跟用户互动，然后逐步引导情绪，让我能有一种积极的，向上的。面对问题解决问题的心态，真的很像一个帮你带出困境的朋友。

情绪控制对我来说是个大问题， 情绪不对，别说效率， 根本启动不起来。weobot在管理情绪的用的方法跟《Get Some Headspace》， 《Headspace冥想正念手册》的方法很像。这本书的作者是Andy Puddicombe， 他有个app叫 headspace。Headspace 让我开始了冥想的第一步。weobot 和heahspace都会引导你把想法和情绪分离开，他们说， 情绪就像高速公路上的车，来了又去，你只要在一旁看着他们， 知道他们来了， 知道他们走了就可以了。 我觉得这个方式蛮好笑的不过也蛮好用的。

当然，心理疗愈是一个循序渐进的过程，我没有办法立竿见影的好起来，但是已经产生了肉眼可见的变化了。我真心的佩服这个制作团队。

工作法

调节了心情，接下来就是行动起来了。方法论这东西， 众说纷纭，或被人吹捧， 或被人唾弃，最终还是要找到自己适合的。我看过不少方法论的书， 大部分时间是为了给自己打点鸡血。今年一月的读书小结（ 1-2月小结–新的轮回）里提到了让我惊喜的书叫《高效PDCA工作术》。不知道什么原因到6月了，我还迟迟没有启动。虽然没有启动，但一直念念不忘，直到我读了《无印良品笔记术：四个步骤实现持续提升》。这本书的日本原名是《無印良品的PDCA》，作者松井忠三是无印良品前会长，他在其最危机的时候接手，一做就是十几年。这本书里没有提到PDCA的细节，大部分的篇幅是在说他自己是怎么用日程表管理来转动PDCA。他在书中直接用自己笔记本的原图，能偷窥到大社长的笔记本也是件幸事。 两本PDCA的书的侧重点不同，前者注重P（计划），后者注重D（行动）。前者让我对PDCA有个很完整的认识， 后者让我的PDCA转动起来了， 把一直以来零碎的周计划日程表也系统起来了。自从每周末开始总结PDCA完成率，每天会不由自主的强迫自己把当天的计划完成。虽然有很多需要完善的地方，终归是找到了适合自己的计划和笔记的方式。

方法有千百种， 没有好坏，要找到自己合适的。

高效PDCA工作术

★★★★☆7.2 （日）富田和成 / 2018 / 湖南文艺出版社

无印良品笔记术：四个步骤实现持续提升

★★★★☆7.5 [日] 松井忠三 / 2019 / 北京时代华文书局

专注力

《为什么精英都有超强专注力》，一本看名字就不想读的书，但在我百无聊赖的时候还是读了， 我很好奇， 为什么我的注意力这么松散，当然这也跟我之前情绪不佳，没有计划有关。但这本书里讲的道理很平常，几乎是被各种公众号写烂的方法。总结一下提供专注力的方法：正常吃饭，按时睡觉，时常运动，坚持冥想，顺其自然，工作细分，调整心态。 但读完后， 我忽然发现这是一本很实用的书， 里面最有让我眼前一亮的方法是找到自己的节律， 日节律， 周节律，月节律和年节律。我对月节律这个深有感触， 一句话被他点醒了，每个月不在状态的那几天， 真的可以放开了心，什么都不想的休息。年节律又对应了上面提到的《无印良品笔记术：四个步骤实现持续提升》的规划方法。这种书时候偶尔读一读， 用外界的刺激，激发一下倦怠的自己。浪费了一个小时，读了这本书之后，以后此类的书我都可以直接跳过了，读再多也没有， 关键要用起来。

为什么精英都有超强专注力

西多昌規 / 2018 / 湖南文艺出版社

艺术史

前一段时间忽然对日本南画感兴趣，偶遇徐小虎。 读过之后，我强烈推荐徐小虎的书。首先她写的中国艺术，日本艺术非常好。好读好懂。做为学术派，能提出自己的见解，而不是一味的罗列知识点，讲解的很清楚。例如她在《南画的形成 : 中国文人画东传日本初期研究》中说日本南画是对中国文人画的变奏，而中国文人画是对古代画作的变奏，我立马就懂了其中的关联。其次有细节，在《被遗忘的真迹 : 吴镇书画重鉴》中，她详细介绍了古物鉴定的细节，并说美术馆50副画，只有3副半的真作！看过之后， 大呼过瘾。

如果说我偏心的话，她真的让我想起来松本清张《真假森林》（收录于《大手笔 》）里的文物鉴定学者，《真假森林》说日本很多权威鉴定师没眼力，又为了讨好达官富贵指鹿为马。一个有真才实学的鉴定师奔走于日本各种穷乡僻壤以寻真迹，惨遭压迫。徐小虎为画奔走，真的有这样的感觉，不同的是她敢讲出来。

南画的形成

★★★★☆7.6 徐小虎 / 2017 / 广西师范大学出版社

被遗忘的真迹

★★★★☆8.4 徐小虎 / 2012 / 广西师范大学出版社

大手笔

★★★★☆8.2 (日)松本清张 著 (日)宫部美雪 导读 / 2012 / 新星出版社

理财

我很喜欢《高效PDCA工作术》这本书， 这书的作者是日本一个金融网站的社长， 年轻有为。我出于好奇读了他的另外一本书《为什么精英可以快速积累财富》。虽然也是又一个很俗气的书名，但书的内容还不错。他用自己的金融专业背景，和PDCA的使用经验，讲解要如何脚踏实地的积累财富，达到自己的目标。读他的书，我发现了自己很多错误的认知，不过“纸上得来终觉浅，绝知此事要躬行“，他的方法我还需要慢慢消化，也许又要一个半年才能循环起来。 另外一本正在读的叫《财务自由之路. Ⅱ 》。前几年读了博多·舍费尔很火的《小狗钱钱 》和《财务自由之路》，对财务自由有了一点儿认识。但是这么多年都完全没有用起来。国内很多理财，财务自由的书恐怕都在抄这个作者。 我从来没有奢望过可以财务自由，但是作者写的很诚恳，至少也可以开阔一下思路。《财务自由之路. Ⅱ 》我才读了前几章，里面的成长性思想跟Woebot 很像。我最近才明白，我的教育里最缺乏的就是成长性思想，需要经常读读这种书来提醒自己。

为什么精英可以快速积累财富

（日）冨田和成 著 博集天卷 出品 / 2019 / 湖南文艺出版社

财务自由之路. Ⅱ

(德) 博多·舍费尔 / 2018 / 现代出版社

数据科学

其他的看的再读的书，不如把专业好好的研究一下。这三个月读了三本。 《Bayesian Reasoning and Machine Learning》是我这学期graphical modeling的教材，写的还不错，不简单也不是特别难懂，。不过（也许）由于疫情的关系， 心不在学习上， 也走马观花的看了一遍，稍微学了一点儿东西。

Bayesian Reasoning and Machine Learning

David Barber / 2011 / Cambridge University Press

《Data Structures and Algorithms in Python》的结构很清晰， 很多字可以跳过直接看图也能理解， 大概两天刷了2/3，把两年前学过的数据结构和算法又复习了一边，顺便把简单的python语句也复习了一遍， 算是入门的佳作。因为有代码夹持， 有非常的实用。我第一次学算法的时候，读的也是本书作者Michael Goodrich 的《Algorithm Design and Applications》，这本也是一本不可多得的好书。他很简单清晰介绍了一下基本和常用的算法。课后习题有三类：常规，创新和应用。应用类习题特别有助于消化理解算法，而且解题过程特别有趣，推荐给需要入门的小伙伴们。唯一不好的就是他P， NP 解释的不好，需要配合其他资源看懂。

Data Structures and Algorithms in Python

Michael T. Goodrich Roberto Tamassia Michael H. Goldwasser / 2013 / John Wiley & Sons

Algorithm Design and Applications

Michael T. Goodrich Roberto Tamassia / 2014 / John Wiley & Sons

《Introduction to Machine Learning with Python》是python-sklearn的使用手册，也是ML非常好的入门之作，但是对知识点的讲解很浅， 很基础也很应用，没有数学和统计的东西， 适合把ML各个知识点学习过之后， 用这本书把知识点都穿起来。我用了不到一天的时间刷完了，把 ML的一些模型， 流程，都过一遍。个人特别喜欢第八章， 没有工业背景的人能了解一点点ML在工业的实用。这本书过一遍就好了，不用花太多时间。

Introduction to Machine Learning with Python

Sarah Guido Andreas C. Mueller / 2016 / O’Reilly Media

轻松一刻

绝代双骄

古龙 / 2013 / 河南文艺出版社

其实上半年大部分的时候都是在休息，读了几本小说。 无意中看了电视剧版的《绝代双骄》勾起了回忆。在海外有的时候就会很想家， 想家的时候就想读一些中国的东西， 看了电视不过瘾又读了书， 读完书之后又太落寞： 一场狂欢曲终人散， 终究一场梦，回到了原点。长大了读起来跟小时候读的感觉就不一样了， 在情节之余看懂了更多人的情绪。

望乡

[日] 凑佳苗 / 2020 / 文治图书·四川文艺出版社

2019年读了好多凑佳苗的书，今年读了《望乡》。我还是很佩服她的笔力的，特别喜欢她第一人称自白式的叙事方法。《望乡》也继承了她以往的风格，她可以把很日常的事情写出一种惊悚感。在第一个故事里，冷不丁的冒出了两个疑问句，寥寥几笔，悬疑就出来了。但是，这本书总体上讲有点无聊。

剩下五本书， 不太出彩， 不提也罢。 总结完上半年，下半年轻装上阵吧。
