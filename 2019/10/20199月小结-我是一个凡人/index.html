2019，9月小结 -- 我是一个凡人

2019-10-1 发布在 读书, 读书小结

意识到这一点以后，心情极度舒畅。很多时候负面情绪来自于self restraint太高。我写的东西没人看我为什么要发出来？我跟他讲话，他不理我怎么办？我的想法太浅薄，是不是会被老板鄙视？诸如此类。既然这样，那我只好搬出我的座右铭，间歇性的提醒一下自己。我是个凡人，plus, like I care.

接下来是极为正常的小结，

推理小说类推荐

混凝土里的金发女郎

[美]迈克尔·康奈利 / 2019 / 北京联合出版公司

我喜欢这个小说的主要原因是它给人的感觉很真实。 这一本很典型的美式推理小说。 确切的说是一本很典型的美国犯罪小说。 这部书读了蛮久的， 大概用了3次才读完。节奏相对比较慢， 没有那种特别抓人的剧情非要一口气读完不可。 因为作者康奈利是一名犯罪新闻记者，所以在叙述办案细节， 法庭争辩，警察处理公务的时候很接近真实，因此读起来不会感觉证据， 推理，剧情都太过于轻浮。故事的主角是个狼狈的警察， 由于误杀了犯人遭到起诉。于是故事在法庭里与法庭外双线进行，两条线的证据相互呼应。喜欢这本书的另外一原因是， 人物塑造的很立体，用戏里戏外的心理战凸显人物性格。唯一的小缺点是作者时不时的释放一些非常细小，若隐若现的线索来“误导”读者。但整体的阅读体验很好。

影之車

松本清張 / 1995 / 星光出版社

松本清张的小众短篇小说集。 包括《潜在影像》，《高雅的姐弟俩》，《万叶翡翠》，《买盆栽的女人》，《淡妆的男人》，《确证》，《乡村医生》。我看清张的几个时期是，看热闹 ->看推理思维 ->看风景 ->看百态人性 -> 看描写手法 -> 看创作逻辑，以及到现在的想去理解作者。前期喜欢长篇小说跟着主角满日本的跑。现在发现短篇越嚼越有味道。喜欢的他的作品的主要原因是有非常充分的动机。我觉得动机才是故事发展的基石。这7篇每篇都有自己的特色， 每篇都像是可能发生在身边的故事，每一篇的人物塑造都栩栩如生。我最喜欢的是《万叶翡翠》，是松本清张典型的考古系推理。三个学生因为以《万叶集》里的一句古诗的推理，进而去野地考察， 随后发生的谋杀故事。看完以后我去google查了小泷川，发现古诗不负我， 真的有翡翠！

非虚构推荐

深度工作

[美] 卡尔·纽波特 / 2017 / 后浪丨江西人民出版社 我承认自我改善类这种仁者见仁的书不需要读的太多，甚至不用读。 但我喜欢间歇性读一下， 给自己打一下鸡血， 有的时候还是有点用的， 例如这一本。这本书主要教你如何闭关，我很喜欢里面到一个与世隔绝的地方， 静静地思考两个月的这种想法；我也很好奇那种彻底放弃社交圈的生活是什么样的。 书的第二部分有适合不同人的一些深度工作的建议， 我试了一下。例如读完书的第一个礼拜，我基本没有怎么碰手机，关闭了邮件， 每天早上起来先心无旁骛的推导公式，发现真的有效！上学期没有一学期都没理解的厉害， 居然被我推导出来了。直到现在， 我也不会在无聊的时候， 排队的时候， 碎片的时候想去看手机了。会不会改善我的工作效率我不知道，但是碎片化实在是害我不浅。 工作学习类推荐

Deep Learning

Ian Goodfellow Yoshua Bengio Aaron Courville / 2016 / The MIT Press 知道这本书的人应该都是在这是一本难得的好书。 我原来以为作者有名气， 被大家捧起来的。读了以后发现， 真的是一本好书。很多半路出家的数据分析者，没有非常扎实的CS基础和数学基础，直接学deep learning 的时候会有很多疑惑的地方。本书的前5章，很详细的介绍了一些必备的基础知识，非常适合新手。

4星推荐类

这一类是我觉得还不错， 但是不够惊艳。

十角馆事件

[日] 绫辻行人 / 2016 / 新星出版社 最近喜欢看暴风雪山庄， 这本书我一口气读完的。跟无人还生相似，也有一些创新的地方。很多暴风雪山庄都是凶杀案都是一天一个的来，这种随机的不知道一下子能死多少个的设定我蛮喜欢的。

斜屋犯罪

★★★★☆7.8 （日）岛田庄司 / 2012 / 新星出版社 非常文艺的推理小说。暴风雪模式+密室模式。结尾很长很长。但是诡计设计的很好，人物的性格设计的也很好，我觉得是他是最像松本清张的本格派了。

密室收藏家

★★★★☆7.9 [日] 大山诚一郎 / 2017 / 江苏凤凰文艺出版社 有很多的梗 ，密室设计的挺不错的，但是有我无法接受的男女混淆以及动机不明确。如果喜欢诡计的话，可以看一下。 三星不推荐类 看的觉得有点浪费时间， 但有不是特别很糟糕的

迷宫馆事件

★★★★☆7.6 [日] 绫辻行人 / 2016 / 新星出版社

水车馆事件

★★★★☆6.7 [日] 绫辻行人 / 2016 / 新星出版社 这两本书太让我失望了，之前度过他的《十字馆》还不错，想顺着把馆系列读完。但是作为第二本的水车馆，和第三部迷宫馆让我没有读下去的欲望了。首先，跟馆一点关系都没有。其次，对于熟悉青村的读者都知道，他的馆一定会有暗道，对于新的读者，就是一种信息不平等。但是基本你知道了会有暗道，但是哪里有暗道是不会给提醒的。所以出现的的密室都有可能不是密室，也都有可能是密室，作者不揭晓答案，谁也不知道。第三，我个人原因，特别不喜欢玩弄字意的诡计。不是说我不喜欢猜谜，男女指代不清，误导性别诡计。

小结

总觉得应该有个结尾。 总结发现9月份没有读很多书，而且主要在看推理小说了。
