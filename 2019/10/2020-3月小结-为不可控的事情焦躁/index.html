2020 3月小结 - 为不可控的事情焦躁

2019-10-1 发布在 读书, 读书小结

整个三月都是焦躁的， 上半月担心国内， 下半月担心自己。没有能力帮不上忙，躲在家里又为何为自己不能控制的事情焦躁？

时间过的真快， 转眼四月了。三月几乎没心情看书，有心情读的时候频繁踩雷。虽然标记了20本书，但读得还不错的书大概只有8本左右。不过还是为了写字的养成习惯， 稍微总结一下。

文学与旅行

太阳与少女

[日]森见登美彦 / 2019 / 湖南文艺出版社

幻想派作家森见登美彦。之前他作品改编的电影《春宵苦短，少女前进吧！》就觉得他脑洞无限。那部乱花迷人眼的作品里的少女充满迷幻般的正能量。 这本书不算小说，是作者发表的一些随笔的合集。作者本着“不会像哲学书那样艰深， 也不像小说那样令人亢奋， 并不是不有趣，却也并非有趣到让人一读就停不下来。“这样的目的集成的。有关于读书，兴趣，闲逛，自著书的衍生，关于作为小说家的经历的多方位讨论。由于作者极度曝光自己的内心深处的想法，加上无下限的调侃自己，而显的十分可爱。一开篇我就被作者笑到了。我也很喜欢作者的坦诚，他探索神隐的经历，他写不出稿的郁闷，他偷窥别人日记的兴奋，他买灵感笔记本的疯狂.. 这些都让我觉得他不是遥不可及的作家，是个活生生的人，可能就是我买菜可以遇到的某一个“猥琐”大叔。我最喜欢整理书房那一节- “漂亮的书要放在鲜艳的位置”，“这里要不动声色的放几本根本不读的哲学书来伪装知性“，”最宝贵的内田百闲的书要放在最好的位置“，“偶像写真要放在来客看不见的位置”… 这种书房整理法则每一点都能踩中我的内心。

除了无限的脑洞和不加掩饰的内心表露，作者的文笔也相当不错。他对文学，电影都有自己独到，新颖的见解。他写出来的东西，好像真的是他日积月累看到的，想到的，感觉到东西。也可以当作日本文学介绍和游记的书来看。 总之，这本书读起来就是舒服，不艰深，不令人亢奋，有趣，但不是一读就停不下来，像是炎热夏天喝到的水蜜桃冷饮，清爽又解渴。

老残游记

刘鹗 / 2016 / 三秦出版社

这本书是个惊喜，因为出乎意料的好看。

起初看题目有老残二字，两个看起来负面没有朝气的词，顿时就没有了兴趣。再看介绍，官场谴责讽刺小说，也不是我平时阅读的范围。这本书一直没有引起我的注意。

之前一段时间准备写蓬莱阁游记，怎么也写不出来 （蓬莱阁这个景点保护不善， 很让我失望）。偶然看到了这段话，老残游记的开场：

“话说山东登州府东门外有一座大山，名叫蓬莱山。山上有个阁子，名叫蓬莱阁。这阁造得画栋飞云，珠帘卷雨，十分壮丽。西面看城中人户，烟雨万家；东面看海上波涛，峥嵘千里。所以城中人士往往于下午携尊挈酒，在阁中住宿，准备次日天来明时，看海中出日。习以为常，这且不表。“

我当时就愣住了。回头想一想蓬莱阁， 在画面中把人头攒动的游客删掉，把庸俗滥造的近几年的建筑删掉，忽然觉得站在阁子上，面向大海，清风徐来，沁人心脾。这是对本书的第一印象：文字太美了， 美到我每个字都想背下来。

第二印象是跟着游记旅行的感觉很棒。第二回和第三回格外出彩。老残“一路秋山红叶，到了济南府”。游玩了一天，看泉听曲。我也不自觉也跟着他“午后便步行至鹊华桥边“，参观铁公祠，或者“吃完茶后， 出了趵突泉后门“ 去寻金线泉。

然而美的东西不足以让我这么喜欢，真的东西画龙点睛。没有故事支撑的游记索然无味。这本书对清朝风土人情，荒谬绝伦的官场故事的描写入骨三分，穿插很多人生态度， 道家佛家的独到见解。总之，看完之后大呼过瘾。

走进大文豪的家

【英】尼克·钱纳 / 2017 / 北京联合出版有限公司

这本书读起来有种满足感。一是像用猫眼偷窥了文豪们的私生活； 二是借着文字力量，去英国旅行了一番，体验了一下宁静，优美，与世隔绝的乡村生活；三是借机让别人帮自己实现一下梦想：文豪房子里满满的都是书。书中印象最深的是阿加莎·克里斯蒂的“圣诞新书会”，那天她会把要新书读给亲友听， 让大家来猜凶手。 这本书我不想一口气读完， 闲来没事读一章，或者去英国玩的时候拿来做攻略用。

编程

Think Python

Allen B. Downey / 2012 / O’Reilly Media

零基础入门级。 如果想从零开始学pyhon，可以从这本书开始。作者事无巨细的把python的基础用法，解剖了讲给读着听。没有什么行业黑话，全部都是行外人能看得懂的语言。我2014年学的python, 多年不用。把这本书用5个小时快速的刷了一遍，刷新了一下记忆，查漏补缺。 如果是其他简单的语言， 例如R，想转到python,也推荐快速扫一下，对比一下不同。如果是如果是C， java 之类的用户， 建议跳过。

Fluent Python

Luciano Ramalho / 2015 / O’Reilly Media

这本书比Think Python 讲的深入很多，可以算作中级吧。花了一天时间看完《Think Python》，再看这本书，觉得之前实在浪费时间，应该直接看这本。 在Python的用法上有很多深入的讨论， 跟其他语言的对比，多行话的解释，编程习惯的建议，以及进一步的推荐阅读。我小打小闹的写python 有一段时间了。很多东西写得出来，但是不够优雅简练。读了这本书会有很多编程的灵感。有了实战经验时候，这本书可以作为阶段性提升的辅助。 不过就是看起来很慢， 差不多1-2个小时才能看完一章。

其他

图解道教

李绿野 / 2008 / 陕西师范大学出版社

很不错的一本道教入门书， 对道教文化解释的算清楚， 全面，客观。 我看完以后有了一个总体的认识， 把平时琐碎的点用问答的方式都串起来了。作者的用词， 怎么说呢，大概是透出了一股仙气。 读到一些典故的时候， 感觉荒谬绝伦。道教不是道家。闲来没事，读着玩玩。
